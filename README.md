# Ary Builds

A static portfolio for documenting hardware projects. Each build gets a hub with specs, parts, wiring diagrams, and a page-flip instruction manual — starting with a face-tracking laser turret.

Built with plain HTML, CSS, and JavaScript. No framework or build step. Animations run on [GSAP](https://gsap.com) (CDN), typography from Google Fonts (Fraunces, DM Sans, JetBrains Mono). All build content lives in `js/data.js`.
